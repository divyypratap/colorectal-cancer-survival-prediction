# Colorectal Cancer Survival Prediction

**Machine Learning Model for Predicting Patient Survival Outcomes**

Built a Logistic Regression classification model on **90,000+ patient records** to predict survival status in colorectal cancer cases. The project covers full data exploration, preprocessing, modeling, and evaluation.

## 📊 Project Overview

This healthcare analytics project aims to predict patient survival based on demographics, lifestyle, and treatment-related variables. The focus was on building an interpretable and practical classification model.

### Key Objectives
- Perform comprehensive Exploratory Data Analysis (EDA)
- Preprocess and encode categorical features
- Build and evaluate a Logistic Regression model
- Identify the most important predictors of survival

## 🔍 Key Insights
- **Stage at Diagnosis**, **Treatment Access**, and **Follow-Up Adherence** are the strongest predictors
- Lifestyle factors like **BMI**, **Smoking Status**, and **Physical Activity** significantly influence outcomes
- The model performed reasonably well after handling class imbalance

## 🛠️ Technologies Used
- Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)
- Logistic Regression with class balancing
- Feature Engineering & One-Hot Encoding
